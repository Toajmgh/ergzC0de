PM2介绍

pm2 是一个带有负载均衡功能的Node应用的进程管理器。当你要把你的独立代码利用全部的服务器上的所有CPU，并保证进程永远都活着，0秒的重载， PM2是完美的。

主要特性：
1.内建负载均衡（使用Node cluster 集群模块）
2.后台运行
3.0秒停机重载，我理解大概意思是维护升级的时候不需要停机
4.具有Ubuntu和CentOS的启动脚本
5.停止不稳定的进程（避免无限循环）
6.控制台检测
7.提供 HTTP API
8.远程控制和实时的接口API(Nodejs模块,允许和PM2进程管理器交互 )
